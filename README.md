# Inventory Management System

A REST API based Inventory Management System built with Django and Django REST Framework.

## 🛠️ Technologies Used
- Python
- Django
- Django REST Framework
- SQLite

## 📋 Features
- Product Management
- Stock Tracking
- REST API Endpoints

## 🚀 How to Run

1. Clone the repository
```bash
git clone https://github.com/kavy123-coder/inventory-management.git
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Run the server
```bash
python manage.py runserver
```
